# FNF Neo R
The full FNF Neo R GitHub repository, ripped straight from the leaked `.git` folder on GameBanana!

Now you can check the whole commit history, (With merge conflicts removed, ofc.) PR to this repo, make Issues for this repo, etc.

Become the change you want to see, and help make the arguable worst FNF mod in history much, much better than how we found it!

(Disclaimer: This GitHub repository is purely fan-made. The original team had nothing to do with the creation of this repository, it was purely my idea to maintain it.)

### Notes about the current direction of the mod
FNF Neo R does not seem to be cancelled, as according to a moderator notice on [its GameBanana page](https://gamebanana.com/mods/644843), the original director of the mod has stepped down, and a user named "LiMACro_OQ" is taking the reins for the project.

It is also no longer an official revival, being labeled as a fan retake from now on.

I would like to contact the new director to talk about this repository, and the future of the mod in general.

Moreover, if any of the original developers want their assets removed from future updates of this repository, then you can file a GitHub issue for that, or contact me personally.

You can contact me at ada_funni on Discord.

# How To Install
> [!WARNING]
> You must install [Codename Engine](https://codename-engine.com) to play FNF Neo R!

1. Add the contents of this repository to a folder named "FNF Neo R".
2. Put the "FNF Neo R" folder into Codename Engine/mods.
3. Launch Codename Engine.
4. Press Tab to open the Mod Picker.
5. Select "FNF Neo R".

If you have any problems with the mod, file a GitHub Issue for them, and I or a community member will try to help you with it.
